Adjustment with Binary Outcomes

Dichotomize the continuous outcome.

data$y_bin <- ifelse(data$y > 2, 1, 0)

Homogeneous working model with simple randomization

glm.homogeneous<-robincar_glm(df = data,
                              response_col="y_bin",
                              treat_col="A",
                              strata_cols=c("z1"),
                              car_scheme="pocock-simon",
                              g_family=binomial(link="logit"),
                              g_accuracy=7,
                              adj_method="heterogeneous",
                              covariate_to_include_strata=TRUE,
                              vcovHC="HC3")

Working model specified using a formula, with biased-coin randomization

glm.formula<-robincar_glm(df = data,
                          response_col="y_bin",

                          strata_cols=c("z2"),
                          car_scheme="biased-coin",
                          g_family=binomial(link="logit"),
